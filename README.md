## 1.2. Setup

Python 3.9.6 or above is required.

```sh
git clone https://github.com/mitchLui/ddcs-scripts.git
cd ddcs-scripts
pip3 install -r requirements.txt
```

---

## 1.3. Usage

### In scripts

As a python script:

```python
import ddcs

ddcs.function(args)
```


### In jupyter notebooks

  
```sh
cd ddcs-scripts
jupyter notebook
```

  
```python
import ddcs
```

NB: Not all functions are shown in the jupyter notebook.
